# ICS Modern WordPress Theme

Production-ready starter block theme for Island Construction Services.

## Install

1. Copy the `ics-modern` folder into `wp-content/themes/`.
2. Activate `ICS Modern` in WordPress admin.
3. Go to Appearance > Editor and set up:
   - Navigation menu links
   - Homepage content blocks
   - Footer quick links
4. Create and publish pages:
   - Home
   - Services
   - Fleet and Equipment
   - Projects
   - Material Orders
   - Request a Quote
   - Contact

## Content model

The theme registers two custom post types:
- `ics_service` for service pages
- `ics_project` for project highlights

## Recommended plugins

- SEO: Rank Math or Yoast SEO
- Forms: Gravity Forms or WPForms
- Performance: LiteSpeed Cache or WP Rocket
- Image optimization: ShortPixel or Imagify

## Notes

- Keep imagery authentic to ICS operations whenever possible.
- Compress photos to WebP and target under 250KB per key image.
- Keep homepage video optional and no autoplay with sound.
