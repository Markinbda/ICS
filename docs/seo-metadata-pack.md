## Technical SEO checklist for launch

- Set canonical URL for each published page.
- Enable XML sitemap and submit to Google Search Console and Bing Webmaster Tools.
- Add LocalBusiness schema with Bermuda address and phone.
- Add Service schema on each service detail page.
- Add FAQ schema only where visible FAQ content exists.
- Optimize all key images with descriptive alt text and WebP format.
- Ensure all pages pass mobile usability and Core Web Vitals thresholds.
- Add internal links between related services and quote/contact pages.
